# Sistema-de-Registro-e-Controle-de-Compras
Sistema de Registro e Controle de Compras feito no Python <br>

O sistema foi desenvolvido em Python com o objetivo de realizar o controle de vendas, permitindo o cadastro de produtos e o cálculo automático de subtotais e do valor total da compra. O usuário pode selecionar a forma de pagamento, com aplicação de descontos automáticos para pagamento à vista, conforme regras pré-definidas. Ao final do processo, o sistema gera um relatório claro e organizado, facilitando a visualização das informações da compra e do valor final a pagar. <br>

Por fim, o sistema gera um relatório detalhado da compra, apresentando de forma organizada:

<ul>
 <li> Lista de produtos cadastrados </li>
 <li> Preço unitário </li>
 <li> Quantidade </li>
 <li> Subtotal por produto </li>
 <li> Forma de pagamento </li>
 <li> Valor total, desconto aplicado e total final a pagar </li>
</ul>
